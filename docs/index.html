Materialize Color Picker

A color picker plugin for Materialize framework

Example

What is it?

The colorpicker allows users to pick a color from a color palette.

Materialize Color Picker is a plugin for Materialize framework (v1.0.0). It allows you to assign it with an input and pick a color as a hexadecimal code to input.
